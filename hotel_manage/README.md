# hotel_manage

- 学期实践项目-酒店管理系统
- 基于flat.ui
- 图标采用svg
- web弹窗插件: layer
- 日期插件：bootstrap-datetimepicker

## keybuild

- `ctrl-/`: 注释
- `ctrl-z`: 撤销
- `ctrl-space`: 自动补全
- `ctrl-shift-v`: md-pre
- `alt-cmd-shift-{`: 折叠所有
- `alt-cmd-shift-}`: 展开所有
- `alt-cmd-[`: 折叠
- `alt-cmd-]`: 展开
- `cmd-k cmd-N`：N为折叠的缩进级别
- `ctrl-alt-cmd-F`:折叠你代码或文本的任意一部分，或者在命令面板中选择“Fold Selection”
- `ctrl-F4`: 关掉当前页
- `ctrl-shift-t`: 恢复关掉的页面
- `alt-shift-t`: todo-show
